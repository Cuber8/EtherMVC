ℹ️ About EtherMVC

What is EtherMVC?

EtherMVC is a secure, lightweight framework for building web applications and websites. It combines modern architecture with simplicity, providing everything you need in a single executable file.

Key Technologies

  • Backend: .NET 6.0 (C#)
  • Frontend: HTML5, CSS3, JavaScript
  • Encryption: AES-256
  • Database: JSON + MySQL support
  • Routing: EtherChemistry
